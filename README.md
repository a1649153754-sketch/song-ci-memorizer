# 宋词背诵辅助网页

一个美观、实用的宋词背诵辅助工具，支持多种记忆模式，并能记录你的背诵进度。

## 功能特点

- **四种背诵模式**：
  - **通读模式**：全文显示，可直接点击字符进行修改纠错。
  - **首字提示**：仅显示每句首字，其余点击后显示。
  - **随机挖空**：随机遮盖30%的字符。
  - **全隐回想**：遮盖所有字符，点击后逐个显示。
- **作者信息**：自动显示词人姓名。
- **记忆追踪**：
  - 点击「标记已背」记录背诵日期。
  - 侧边栏显示背诵进度和已完成标记（✓）。
  - 数据存储在本地浏览器（LocalStorage）。
- **精美设计**：深色调玻璃拟态风格，适配多种屏幕。

## 本地运行

1. 下载此项目。
2. 在项目根目录下启动一个静态服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器打开 `http://localhost:8000`。

## 文件结构

- `index.html`: 页面结构。
- `styles.css`: 样式文件。
- `app.js`: 核心交互逻辑。
- `data.js`: 宋词数据源。
- `*.py`: 数据提取与处理脚本。
